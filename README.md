# SQL Project 1: Mobile Prices Analysis (Source : Kaggle)
This repository contains SQL scripts used for analyzing sales data.
The dataset was exported from MySQL Workbench and processed using basic SQL operations such as filtering (WHERE, GROUP BY, ORDER BY, HAVING )and aggregation.
Coloumns in this Data including Brand, Model, Price, RAM, Storage, Camera, Battery, Display Size, Charging Watt, 5g Support, Processor, Realease Month, and Year 

# SQL Project 2 : Product Order Analysis/ SQL_VOL 1  (Source : dummy data)
This repository contains SQL scripts used for analyzing Product and Costumer data.
The dataset was exported from MySQL Workbench and processed using basic- intermediate SQL operations
such as filtering (WHERE, ORDER BY, GROUP BY, HAVING), JOIN, aggregation, Case, Common Table Expession (CTE) (WITH), Window Functions – Ranking Functions (Dense_Rank (), Rank () OVER), Window Functions – Aggregare Functions (SUM/COUNT/AVG () OVER)
